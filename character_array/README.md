# Character Array

Character array is an array where characters are stored in each index.

```mermaid
graph LR;
    A[A]---B[B];
    B---C[C];
    C---D[D];
    D---E[E];
    E---F[F];
```

## Declaration of Character arrays

Character Arrays are declared similar to integer arrays but with an additional 1 byte size.
